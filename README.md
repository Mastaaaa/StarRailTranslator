# StarRailTranslator

This is an italian translator for the game Honkai: Star Rail. It doesnt read or write any process memory. It uses OCR to retrieve text from game screenshots. The text is translated using Deepl API.
For personal use, u need to create ur own Deepl API key, than create a .env file with this code line ---> API_KEY="YOUR_KEY". The .env file must be inside the StarRailTranslator folder.
